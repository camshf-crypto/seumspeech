여기에 사진을 넣으세요.

src/config.js 의 IMAGES 에 적은 파일명과 똑같이 넣으면 됩니다.

예시)
  /public/images/hero.jpg      ← config.js 에서  heroBg: "/images/hero.jpg"
  /public/images/course1.jpg   ← course1: "/images/course1.jpg"
  /public/images/logo.png      ← logo: "/images/logo.png"

권장 사이즈
  - heroBg (히어로 배경): 1920 x 1080
  - course1~3 (강좌)     : 600 x 400
  - review1~4 (후기)     : 500 x 350
  - gallery (배움터)     : 800 x 600
  - logo (로고)          : 가로 200px 내외, 배경 투명 PNG
